### 题目描述

一只青蛙一次可以跳上1级台阶，也可以跳上2级……它也可以跳上n级。求该青蛙跳上一个n级的台阶总共有多少种跳法。

### 思路

依题意，可推出递推公式为```F(N) = F(N - 1) + F(N - 2) + F(N - 3) + .....+ F(1)```；将```N - 1```代入，可得到```F(N - 1) = F(N - 2) + F(N - 3) + .....F(1)```。

将两式相减得，```F(N) = 2 * F(N - 1)```

### C++代码实现

```c++
int jumpFloorII(int number) {
    int first = 1;
    for(int i = 1; i < number; i ++)
    {
        first = first * 2;
    }
    return first;
}
```
